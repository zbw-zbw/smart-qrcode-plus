## 🚀 部署使用

### 快速开始
```bash
# 克隆项目
git clone https://github.com/zbw-zbw/smart-qrcode-plus.git
cd smart-qrcode-plus

# 安装依赖
npm install

# 构建扩展
npm run build

# 在Chrome中加载dist目录
```

### 开发模式
```bash
# 启动开发服务器
npm run dev
# 在Chrome扩展页面加载项目根目录
```
